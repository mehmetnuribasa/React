on CMD
creating a file:    touch (GitBash)
                    echo. > example.txt(CMD)

creating a folder:  mkdir

removing a file:    del example.txt
                    del /f example.txt  (If the file is read-only, force delete it:)

removing a folder:  rmdir example
                    rmdir /s example    (Deletes the folder and everything inside it.)

To pushing your repositorry to your GitHub, create a new repository on github then, follow steps

git branch -M main
git remote add origin https://github.com/mehmetnuribasa/myApp.git   (add the remote repo address by "origin" to your local repo. It provides the connection)

git push -u origin main                 (push your local changes to remote repo)
git push                                (this is enough for later pushes)
